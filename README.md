hueCli (unstable)
======

Command line interface for philips hue. this very early version will be refactored a lot.

# Install

## via npm (recommended)

```bash
npm install -g huecli
```

# Usage

```bash
# activate test scenes
hue scene on
hue scene colorloop
hue scene off

# notify command flashes your lights once
hue notify
# or for 5 seconds
hue notify -d 5
```
