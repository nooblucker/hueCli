hueCli (unstable)
======

Command line interface for philips hue. this very early version will be refactored a lot.

# Issues

the git style subcommand causes problems: https://github.com/visionmedia/commander.js/pull/224

# Install

## via npm (recommended)

```bash
npm install -g huecli
```

# Usage

```bash
# activate test scene
hue scene colorloop
```
